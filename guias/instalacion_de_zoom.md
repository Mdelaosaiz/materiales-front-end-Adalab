# Instalación de VS Code

### Zoom desde Ubuntu

- Accede al instalador de aplicaciones desde el menú
- Buscar **Zoom** e instálalo.

### Zoom desde Windows

- Descarga e instala [Zoom](https://zoom.us/support/download).

### Zoom desde Mac

- Descarga e instala [Zoom](https://zoom.us/support/download).
